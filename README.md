# Phishing Awareness Simulation

## Description
This project demonstrates a phishing awareness simulation using a fake login page and email scenario. It helps in understanding how phishing attacks target users and how people respond to such situations.

## Objective
- To study phishing techniques
- To simulate a phishing attack
- To observe user behavior
- To improve awareness about cyber threats

## Technologies Used
- HTML
- CSS
- JavaScript

## Working
A simulated phishing email is created with a link to a fake login page. When users enter details, the data is stored locally for analysis. This helps in understanding how users react to phishing attempts.

## Key Points
- Focus on cybersecurity awareness
- Demonstrates social engineering concept
- Safe and controlled environment

## Disclaimer
This project is created only for educational purposes. No real data is collected or misused.
